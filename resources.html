Research

Equipping researchers with the machine learning skills

Through a range of learning and development activities, Accelerate Science is equipping researchers with the machine learning skills and tools that will enable them to drive a new wave of scientific progress. This page brings together lectures, software and code releases, and other resources that can help researchers use machine learning to advance their science.
